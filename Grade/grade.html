Systems_BSI • Grade Oficial

Aulas Semanais de Criação de Bots (Python & JavaScript)

Programa contínuo de formação em bots para Discord, do nível iniciante ao avançado, usando Python, Node.js e o ecossistema oficial do Discord.

Domingos • Aulas semanais Formato prático e guiado Foco em bots reais para o servidor

▸ Módulo 1 – Primeiros Passos (Iniciantes)

Aula 1 – Criação do Bot (Primeiro Bot)

Ponto de partida

Objetivo: Ensinar a criar o primeiro bot usando o Development for Discord, entender o que é um bot e como ele se conecta ao servidor.

  • O que é um bot do Discord e como funciona
  • Criando o bot pelo painel de aplicações do Discord
  • Configurando intents, permissões e escopos
  • Gerando o link de convite e adicionando o bot no servidor
  • Criando e testando o primeiro comando básico

Aula 2 – Estrutura de Comandos (Slash Commands)

Fundamentos

Objetivo: Entender como funcionam os slash commands, organizar comandos do bot e criar uma base sólida para todas as próximas aulas.

  • Criação de comandos básicos: /ping, /help, /info
  • Diferença entre comandos globais e de guild
  • Respostas públicas x respostas privadas (ephemeral)
  • Organização de comandos por categorias

Aula 3 – Eventos Básicos

Reatividade

Objetivo: Apresentar o conceito de eventos e fazer o bot reagir automaticamente a ações que acontecem no servidor.

  • Eventos de entrada e saída de membros
  • Eventos de mensagens (on message)
  • Logs simples via console
  • Boas práticas para não “spammar” o servidor

▸ Módulo 2 – Interações e Funções Úteis

Aula 4 – Embeds e Mensagens Estilizadas

Visual

Objetivo: Criar respostas mais profissionais, usando embeds, imagens e componentes interativos.

  • Criação de embeds com títulos, descrições, cores e campos
  • Uso de imagens, thumbnails e footer
  • Introdução a botões e menus de seleção

Aula 5 – Sistema de Registros e Logs

Organização

Objetivo: Ensinar o bot a registrar ações importantes em canais de log.

  • Canal de logs para comandos utilizados
  • Registro de entradas/saídas de membros
  • Estrutura básica para outros tipos de logs

Aula 6 – Criando Sistemas Personalizados

Aplicação prática

Objetivo: Construir funções úteis focadas no servidor da comunidade.

  • Sistema de boas-vindas personalizado
  • Atribuição automática de cargos em condições simples
  • Comandos de informações gerais do servidor

▸ Módulo 3 – Bots Inteligentes e Integrações

Aula 7 – Conexão com APIs Externas

Integrações

Objetivo: Ensinar o bot a buscar informações fora do Discord (internet, APIs públicas).

  • Consumo de APIs simples (clima, moedas, etc.)
  • Criação de comandos como /clima e /preço
  • Tratamento básico de erros ao chamar APIs

Aula 8 – Mini Banco de Dados (JSON)

Persistência simples

Objetivo: Mostrar como salvar informações em arquivos locais usando JSON.

  • Estrutura de dados em JSON
  • Sistema de economia/XP simples
  • Leitura e escrita de dados para cada usuário

Aula 9 – Banco de Dados Profissional (SQLite/MongoDB)

Escala

Objetivo: Dar o próximo passo usando bancos de dados reais, preparados para bots maiores.

  • Conceitos básicos de SQLite e/ou MongoDB
  • Criação de tabelas/coleções para usuários
  • Sistema de níveis persistente

▸ Módulo 4 – Projetos Avançados

Aula 10 – Dashboard Web do Bot

Fullstack

Objetivo: Conectar o bot a uma interface web simples para visualização de dados.

  • Página web básica (HTML/JS) integrada ao bot
  • Exibição de status, comandos, métricas simples

Aula 11 – Criação de um Bot Completo

Projeto final

Objetivo: Desenvolver um bot completo escolhido em conjunto com os alunos (moderação, economia, eventos, etc.).

  • Definição do tipo de bot
  • Planejamento de funcionalidades
  • Implementação guiada em aula

Aula 12 – Publicação, Hospedagem e Segurança

Produção

Objetivo: Mostrar como deixar o bot online 24/7 de forma segura.

  • Hospedagem em serviços como Railway/Render/VPS
  • Uso de variáveis de ambiente para proteger o token
  • Boas práticas, manutenção e atualizações
